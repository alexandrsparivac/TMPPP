# Scripts

Acest folder pastreaza utilitarele folosite in dezvoltare, separate de codul aplicatiei.

- `patches/` - scripturi istorice de patch pentru modulele principale.
- `fixes/` - scripturi de reparare sau migrare punctuala.
- `dev/` - experimente si verificari rapide.

Ruleaza scripturile din radacina proiectului, deoarece multe dintre ele folosesc cai relative catre `student_life_helper/`.
